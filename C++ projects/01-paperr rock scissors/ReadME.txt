# Paper Rock Scissors Game

This is a simple console-based game of Paper Rock Scissors created using C++. It's designed for educational purposes to learn basic programming concepts.

## How to Play

1. Clone or download the repository.
2. Compile the source code using a C++ compiler.
3. Run the executable file generated.
4. Follow the on-screen instructions to play the game:
   - Choose your move (paper, rock, or scissors) by entering the corresponding number.
   - The computer will randomly select its move.
   - The winner of the round is determined based on the classic game rules:
     - Paper covers rock
     - Rock smashes scissors
     - Scissors cut paper
5. Repeat steps 4 to play additional rounds.
6. The game will keep track of the score.
