## Features

- User authentication using account number and PIN.
- Withdrawal: Allow users to withdraw funds from their account.
- Deposit: Allow users to deposit funds into their account.
- Balance Inquiry: Allow users to check their account balance.
- Account Management: Ability to create new accounts and update account details (for admin mode).
- Multi-account support.

## How to Use

1. Clone or download the repository.
2. Compile the source code using a C++ compiler.
3. Run the executable file generated.
4. Follow the on-screen instructions to use the ATM system:
   - Enter your account number and PIN to log in.
   - Choose from the available options (withdrawal, deposit, balance inquiry, etc.).
   - Follow the prompts to complete your transaction.
5. Log out when finished.
