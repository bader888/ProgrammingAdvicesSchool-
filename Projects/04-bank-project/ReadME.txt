# Bank Management System with Currency Exchange

This is a comprehensive console-based Bank Management System project created using C++. 
It simulates various banking functionalities such as account management, 
transaction handling,  currency exchange, and more.

## Features

- Account Management: Create, update, and delete customer accounts.
- Transaction Handling: Perform various transactions including deposits, withdrawals, transfers, and balance inquiries.
- Currency Exchange: Convert between different currencies based on exchange rates.
- Multi-user Support: Ability to manage multiple user accounts and their transactions simultaneously.
- Admin Mode: Access admin functionalities for managing system settings, user permissions, and more.

## How to Use

1. Clone or download the repository.
2. Compile the source code using a C++ compiler.
3. Run the executable file generated.
4. Follow the on-screen instructions to use the Bank Management System:
   - Log in as a user or an admin.
   - Choose from the available options to perform various banking operations including currency exchange.
   - Follow the prompts to complete transactions and manage accounts.
5. Log out when finished.
