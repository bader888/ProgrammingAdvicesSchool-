# Basic Math Game

This is a simple console-based math game created using C++. It's designed for educational purposes to practice basic arithmetic operations.

## How to Play

1. Clone or download the repository.
2. Compile the source code using a C++ compiler.
3. Run the executable file generated.
4. Follow the on-screen instructions to play the game:
   - Choose the type of arithmetic operation you want to practice (addition, subtraction, multiplication, division).
   - The game will generate random math problems based on your selection.
   - Enter your answer for each problem.
   - The game will provide feedback on whether your answer is correct or incorrect.
5. Repeat steps 4 to practice more math problems.
